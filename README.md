# Juan Carlos Tovar Gómez - Portfolio

Personal portfolio website for Juan Carlos Tovar Gómez, Head of Technological Development Department at FES Iztacala, UNAM.

## About

This is a professional portfolio showcasing expertise in:
- Learning Management Systems (LMS)
- Distance Education
- Educational Technology
- System Development
- ICT Training
- Digital Analytics

## Features

- **Responsive Design** - Fully responsive layout optimized for desktop, tablet, and mobile devices
- **Smooth Animations** - Scroll-triggered fade-in animations and parallax hero section
- **Professional Theme** - Medieval-inspired design with gold accents and elegant typography
- **Single Page** - Fast-loading static HTML with no build dependencies

## Technology Stack

- HTML5
- CSS3 (Grid, Flexbox, Custom Properties, Animations)
- Vanilla JavaScript (no frameworks)

## Deployment

This site is hosted on GitHub Pages at [jctovar.github.io](https://jctovar.github.io)
